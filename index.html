      ☁️ Sincronización con GitHub

      Los datos del semanario se guardan como un fichero JSON dentro de un repositorio de GitHub. Cada cambio queda registrado como un commit y los 4 dispositivos ven lo mismo al recargar.

      Cómo obtener el Personal Access Token (PAT)
      1. Entra en github.com/settings/tokens (Fine-grained tokens)
      2. Generate new token. Nombre: "Semanario". Expiración a tu gusto.
      3. En Repository access elige Only select repositories y selecciona el repo donde estarán los datos.
      4. En Repository permissions → Contents: Read and write.
      5. Generate token, copia el valor (empieza por github_pat_…) y pégalo abajo.
      6. ⚠️ El token se guarda solo en este dispositivo (localStorage). Cada miembro de la familia usa su propio token.
